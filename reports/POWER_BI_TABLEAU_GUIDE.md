# Power BI / Tableau Dashboard Guide

Use the CSV files from `exports/` as dashboard sources.

## Page 1: Placement Overview

Source files:

- `kpi_summary.csv`
- `yearly_trend.csv`

Recommended visuals:

- Card: total students
- Card: placed students
- Card: placement rate
- Card: average package
- Line chart: placement rate by year
- Column chart: placed students by year

## Page 2: Department Analysis

Source file:

- `department_summary.csv`

Recommended visuals:

- Bar chart: placement rate by branch
- Matrix: total students, placed students, average CGPA, average package
- Conditional formatting: placement rate

## Page 3: Company Analytics

Source file:

- `company_summary.csv`

Recommended visuals:

- Bar chart: top recruiters by hires
- Table: company, industry, role, average package
- Scatter chart: hires vs average package

## Page 4: Package Analytics

Source file:

- `package_distribution.csv`

Recommended visuals:

- Donut chart: students by package band
- Bar chart: average package by package band

## Page 5: Skill Analytics

Source file:

- `skill_demand.csv`

Recommended visuals:

- Bar chart: recruiter demand by skill
- Bar chart: average student proficiency by skill
- Calculated view: skill gap

## Page 6: Student Readiness

Source file:

- `student_readiness.csv`

Recommended visuals:

- Table: student readiness score
- Filter: branch
- Filter: status
- Bar chart: readiness category count

## Suggested Dashboard Title

Placement Analytics and Career Insights System Using Data Analysis and Business Intelligence
